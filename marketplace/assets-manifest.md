## Screenshot Guidelines

### Homepage Screenshot
- Show clean interface with URL input
- Include "Multi-Page Crawl Now Available" badge
- Show feature highlights

### Audit Results Screenshot
- Display overall score prominently
- Show category breakdowns (SEO, Performance, etc.)
- Include multi-page analysis section

### PDF Report Screenshot
- Show first page of PDF (cover)
- Display professional formatting
- Include company branding

### White-Label Screenshot
- Show customization panel
- Display custom colors/logo applied

## Video Production Tips

### Tools Recommended
- **Loom** (free) - Quick screen recording
- **Camtasia** ($249) - Professional editing
- **OBS Studio** (free) - Recording + streaming
- **Figma** - For lower thirds and graphics

### Best Practices
1. Use 1080p resolution
2. Add background music (royalty-free)
3. Include captions/subtitles
4. Show cursor for navigation
5. Keep under 60 seconds
6. Add end card with CTA
